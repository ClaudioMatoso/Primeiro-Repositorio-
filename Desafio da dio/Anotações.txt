Link para baixar o git 

codigo usados dentro do git que são importante

git clone (este comando serve para clonar o que tem dentro do github para dentro da sua pasta em seu computador)

cd (nome do repositorio) (este comando serve para entrar dentro de uma pasta)

git status (serve para ver se houve alguma alteração dentro dos arquivos ou pastas)

git add . ou git add -A (Serve para considerar o novo documento para o controle de versão)

git commit 

git push origin main (serve para mandar os commit locais para a nuvem )
